# Inkspan

Inkspan is open metadata infrastructure for comics and manga.

We’re building a clean, structured catalog layer that other tools can build on, including apps, libraries, search tools, readers, and discovery platforms.

## Projects
- `api` — public API for metadata lookup and retrieval
- `worker` — ingest, normalization, and indexing jobs
- `schema` — canonical database schema and migrations
- `docs` — documentation and API references
- `ops` — deployment, backups, and infrastructure

## Goals
- stable IDs
- multilingual titles
- structured releases and credits
- product-level identifiers and pricing
- text-to-ID resolution
- infrastructure-first design
